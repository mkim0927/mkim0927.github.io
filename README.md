# Researcher GitHub Pages Template

개인 연구자용 GitHub Pages 템플릿입니다. 논문, 연구 주제, 프로젝트, 강의/멘토링, 연락처 섹션을 포함합니다.

## Publish

1. GitHub 저장소의 `Settings`로 이동합니다.
2. `Pages`에서 배포 소스를 `Deploy from a branch`로 설정합니다.
3. 브랜치는 `main`, 폴더는 `/root`를 선택합니다.
4. 저장하면 `https://<username>.github.io/<repository>/` 형태로 배포됩니다.

## Customize

- `index.html`: 이름, 소개 문구, 논문, 프로젝트, 링크를 수정합니다.
- `styles.css`: 색상, 간격, 타이포그래피를 조정합니다.
- `assets/research-hero.png`: 첫 화면 배너 이미지입니다.
- `script.js`: 논문 필터 같은 작은 상호작용을 담당합니다.

## Image Prompt

The hero image was generated with the built-in image generation tool using a researcher portfolio prompt: modern research desk, annotated papers, subtle data visualization screens, glass slides, and a clean computational network motif, with no logos, no readable text, and no watermark.
